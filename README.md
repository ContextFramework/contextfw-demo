Context Framework Demo
======================

The purpose of this project is to provide an example that is using different 
techniques to demonstrate what the framework is capable doing.

What is done
------------

 1. Basic code usage
 1. Minifier
 1. Asynchronous tasks
 1. Cloud-capabilities

What is in progress
-------------------

 1. Localization demo
 1. JsTemplates demo
